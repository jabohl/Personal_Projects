## 🎯 Projects

### 1. 🧮 [Calculator](./Calculator/)

A modern, fully functional arithmetic calculator built with vanilla JavaScript.

**Features:**
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clear and delete functions
- Keyboard support for fast calculations
- Responsive design for mobile and desktop
- Beautiful gradient UI with smooth animations
- Error handling for invalid operations

**Tech Stack:** HTML5, CSS3, Vanilla JavaScript

**Status:** ✅ Complete

---

### 2. 🎮 [Tic Tac Toe](./TicTacToe/)

A fully functional two-player Tic Tac Toe game with win detection and statistics tracking.

**Features:**
- Two-player gameplay (X and O)
- Automatic win detection (rows, columns, diagonals)
- Draw detection
- Game statistics tracking
- Reset game functionality
- Responsive and mobile-friendly design
- Smooth animations and modern UI

**Tech Stack:** HTML5, CSS3, Vanilla JavaScript

**Status:** ✅ Complete

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (for viewing/modifying code)

### Running Projects

Each project can be run independently:

1. Navigate to the project directory
2. Open the `index.html` file in your web browser
3. No installation or build process required

```bash
# Example: Running the Calculator
cd Calculator
open index.html  # macOS
# or
start index.html  # Windows
```
