# Simple Arithmetic Calculator

A modern, fully functional arithmetic calculator built with HTML, CSS, and JavaScript.

## Features

- ➕ **Basic Operations**: Addition, subtraction, multiplication, and division
- 🎯 **Clear Functions**: AC (All Clear) and DEL (Delete Last)
- ➖ **Sign Toggle**: Switch between positive and negative numbers
- 🔢 **Decimal Support**: Full support for decimal numbers
- ⌨️ **Keyboard Support**: Use your keyboard for faster calculations
- 📱 **Responsive Design**: Works perfectly on desktop and mobile
- 🎨 **Modern UI**: Beautiful gradient design with smooth animations
- 🛡️ **Error Handling**: Prevents division by zero and invalid operations
- 📊 **Calculation History**: Tracks all calculations during session

## Files

- `index.html` - Calculator structure and layout
- `styles.css` - Modern styling with gradients and animations
- `calculator.js` - Complete calculator logic
- `README.md` - Documentation

## How to Use

1. Open `index.html` in your web browser
2. Click number buttons to input values
3. Click an operation button (+, -, ×, ÷)
4. Enter the second number
5. Click the equals button (=) to calculate
6. Use AC to clear everything or DEL to delete the last digit

## Button Functions

| Button | Function |
|--------|----------|
| **0-9** | Enter numbers |
| **.** | Decimal point |
| **+** | Addition |
| **-** | Subtraction |
| **×** | Multiplication |
| **÷** | Division |
| **=** | Calculate result |
| **AC** | Clear all (reset calculator) |
| **DEL** | Delete last digit |
| **+/-** | Toggle sign (positive/negative) |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| **0-9** | Enter numbers |
| **.** | Decimal point |
| **+, -, \*, /** | Operations |
| **Enter** or **=** | Calculate |
| **Backspace** | Delete last digit |
| **Escape** | Clear all |
